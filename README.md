# 🚢 Titanic Survival Predictor

## 🎯 Problem Statement
Predict passenger survival using ML — deployed as a live interactive web app.

## 🔍 Key Results
- 83% Accuracy | 0.897 ROC-AUC
- 4 engineered features improved baseline by 2-3%
- Outperformed Logistic Regression baseline

## 🛠️ Tech Stack
Python | Pandas | Scikit-learn | Random Forest | Streamlit

## 📊 Key Insights
- Women had 74% survival rate vs 19% for men
- 1st class passengers: 63% survival vs 24% in 3rd class
- Title engineering (Mr/Mrs/Miss) was the strongest predictor
